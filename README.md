# 📚 Concepts Covered

## 🎯 Selectors & Targeting

### 🔹 Attribute Selectors
- Target elements based on attributes
- Used for dynamic styling without extra classes

```css
input[type="text"] {
  border: 2px solid blue;
}
```

---

### 🔹 Pseudo Classes
- `:hover`
- `:focus`
- `:active`
- `:first-child`
- `:last-child`

Used for interactive UI behavior.

---

### 🔹 Pseudo Elements
- `::before`
- `::after`
- `::first-letter`
- `::first-line`

Enhancing UI without extra HTML elements.

---

### 🔹 nth-child()
Precise element targeting inside parent containers.

```css
li:nth-child(odd) {
  background-color: #f2f2f2;
}
```
